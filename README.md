# Real-Time Character Counter

A simple and responsive **Real-Time Character Counter** built using **HTML, CSS, and JavaScript**. This project counts the number of characters typed in a text area and allows users to enter **up to 200 characters only**. Once the limit is reached, the application displays a **"Limited Access Denied"** message and prevents additional input.

## 🚀 Features

* Real-time character counting
* Maximum limit of **200 characters**
* Displays remaining characters
* Shows **"Limited Access Denied"** when the limit is exceeded
* Prevents typing beyond 200 characters
* Clean and responsive user interface
* Built with HTML, CSS, and JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

